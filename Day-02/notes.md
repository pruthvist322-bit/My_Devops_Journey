# Day 2 - Linux Fundamentals

## Topics Learned

### What is Linux?
Linux is open source operating system kernel used to manage hardware and software resources.

### Technologies before Linux?
Unix --> which was introduced in 1961 by Ken Thompson and Dennis Ritchie and it is not open source resource and not everyone could afford it and facing a compatiblity issue.
GNU Project ---> which was introduced in 1980 by Richard Stallman which was open soucrce but it was  facing a kernel problem.
Linux/GNU Linux --->Which was introduced by Linus Torvalds which was open source where he build a linux kernel from scratch by using GNU tools.

### Advantages of Linux?
- More reliable
- Multiuser support
- Free and open source
- Customizable
- Flexible
- Secure
- Stable

### Linux Boot Process
