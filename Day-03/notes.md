## Creating an AWS Free Tier Account
### Step 1: Open AWS Website
#### Visit:
https://aws.amazon.com/
You will see:
#### Create an AWS Account
#### Sign In
## Click on Create an AWS Account.

### Step 2: Enter Root Email Address
#### AWS asks for:
- Root email address
- AWS account name

Example:
Account Name: Dev1
## Then click on Verify email address.

###  Step 3: Verify Email
AWS sends a verification code to the registered email address.
Steps:
1) Open the email
2) Copy the verification code
3) Enter the code in AWS
4) Verify the account
##
### Step 4: Fill Personal Information
AWS asks for:
- User name
- Address
- Locality
- City
- Pincode
Then allow AWS permissions by selecting the required checkbox/radio button.
 ### Click Next.
##
### Step 5: Payment Information
AWS asks for:
Debit card / Credit card details

Required details:
- Card Number
- CVV
- Validity
- Card holder name
- Mobile number

Fill the details and click Next.
##

### Step 6: Account Purpose & DigiLocker Verification
AWS asks for account type.
I selected:
- Individual
Then AWS asks for DigiLocker verification.
You can use:
- Root email owner's DigiLocker
  or
- Card holder's DigiLocker
depending on the situation.
##
### Step 7: OTP Verification
- AWS charges an initial verification amount (around ₹2).
- The amount will usually be credited back later.
- After OTP verification and successful payment verification, the AWS account gets created.
- AWS Free Tier also provides free credits for learning purposes.
##

## Problems Faced During AWS Account Creation
1. Email Already Used
- AWS does not allow reusing an email already linked to another AWS account.
### Solution
- Use a new email address that has not been used earlier for AWS account creation.
##
2. Card Payment Rejection
- Sometimes debit cards may fail during verification.
### Solution
- Using a Visa card may make the process smoother.
##
3. DigiLocker Verification Issues
- Verification may fail if login details are unavailable.
### Solution
- Keep DigiLocker ID and password ready before starting the process.
##
