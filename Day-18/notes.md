# Day-18 | Introduction to Shell Scripting

## Overview

Today I learned the fundamentals of Shell Scripting and how automation works in Linux.

Shell scripting allows us to execute multiple commands automatically using script files.

---

## Topics Covered

### What is Shell

A shell is an interface between the user and the operating system.

### What is Bash Scripting

Bash scripting allows writing commands inside a file and executing them together.

### What is Automation

Automation helps execute repetitive tasks automatically.

---

## Shebang

```bash
#!/bin/bash
```

Used to invoke Bash shell.

If not specified, default shell will be used.

---

## Useful Commands

```bash
cat /etc/shells
```

View available shells.

```bash
echo $SHELL
```

Check default shell.

```bash
echo $PATH
```

Check default path.

```bash
echo $USER
```

Check current user.

---

## Running Scripts

```bash
./scriptname
sh scriptname
bash scriptname
```
