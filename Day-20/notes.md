## Day 20 – Introduction to Cloud Computing

Today I completed my Linux fundamentals learning and started learning Cloud Computing concepts.

Topics Covered:

• What is Cloud Computing
Cloud Computing is the delivery of computing services such as servers, storage, networking, databases, and software over the internet instead of managing physical infrastructure.

---

Difference Between On-Premises and Cloud Computing

On-Premises:

* Infrastructure is managed locally
* Higher initial investment
* Maintenance handled by organization
* Limited scalability

Cloud Computing:

* Resources provided over internet
* Pay-as-you-use model
* Highly scalable
* Reduced infrastructure management

Advantages of Cloud:

* Scalability
* Cost Optimization
* High Availability
* Global Accessibility
* Faster Deployment

---

Cloud Service Models

1. Infrastructure as a Service (IaaS)

* Provides virtual infrastructure
  Examples:
* Virtual Machines
* Storage
* Networking

2. Platform as a Service (PaaS)

* Provides platform to develop and deploy applications

3. Software as a Service (SaaS)

* Complete software delivered through internet

---

Cloud Deployment Models

Public Cloud

* Shared infrastructure
* Accessible through internet

Private Cloud

* Dedicated infrastructure for single organization

Hybrid Cloud

* Combination of Public and Private cloud

Key Learning:

Cloud Computing reduces infrastructure management and enables organizations to build scalable and highly available applications.
